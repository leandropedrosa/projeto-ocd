# OCD Backend

## Configuração do projeto (Base de dados e Build)
# - Primeiro passo: Base de dados:

- Base de dados: Criar a base no MongoDB: `use ocd-DB`

# - Segundo passo: Build:
- Maven: `mvn clean package`
- Spring-boot: `mvn spring-boot:run`
